# Collect Audio Streaming

Developing **Proof of Concept (PoC)** for project **MEC-ITA** using Streaming Technology.

## Front-End

Functional Requirements to Front-End

| Requitements ID  | Description  |
|---|---|
|  RF1 | Captar Áudio  |
